Projeto  Todo List!!

Este projeto tem como objetivo simular uma lista de tarefas, onde o usuario  pode escrever o que precisa e depois que concluir
pode excluir.

Tecnologias utilizadas
NextJS
Javascript
CSS
Github
Deploy realizado na Vercel
